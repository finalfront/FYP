# FYP: Green Synthesis Route Evaluator

An LLM-powered framework that evaluates and ranks retrosynthetic routes based on their environmental sustainability using the **12 Principles of Green Chemistry**.

## Overview

This project leverages large language models to assess chemical synthesis pathways and score them against established green chemistry principles, helping chemists identify more sustainable synthetic routes.

## Features

- 🔬 Retrosynthetic route analysis
- 🌱 Sustainability evaluation using Green Chemistry principles
- 🤖 LLM-powered assessment and ranking
- 📊 Comparative route analysis

## The 12 Principles of Green Chemistry

1. Prevention
2. Atom Economy
3. Less Hazardous Synthesis
4. Designing Safer Chemicals
5. Safer Solvents and Auxiliaries
6. Design for Energy Efficiency
7. Use of Renewable Feedstocks
8. Reduce Derivatives
9. Catalysis
10. Design for Degradation
11. Real-time analysis for Pollution Prevention
12. Inherently Safer Chemistry for Accident Prevention
